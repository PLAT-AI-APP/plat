# 도메인 설계 가이드

## 사용자 요구사항

### User

#### 회원 유형
- 이메일
  - 이메일 - 본인인증
  - 비밀번호 - 8자 이상 특수문자 포함
- 소셜로그인
    - 구글
    - 카카오톡
이하 이메일 또는 소셜로 로그인한 회원을 '**로그인유저**'라 함

#### 회원가입
* 최초 로그인한 로그인유저는 부가정보를 입력해야 최종 회원가입이 가능하다.

- 닉네임*
    - 1 ~ 15자 까지 허용
    - 특수문자 불가
    - 중복 불가
- 생년월일*
    - 만14세가 넘어야 됨
- 성별*
    - 남자, 여자, 비공개로 구분

## 회원탈퇴
동일한 계정 또는 휴대폰으로 7일간 재가입 불가
