1. 문단 구분은 "\n\n" 으로한다.
2. 문단에는 다음과 같은 종류가 있다.
    - 상황설명(시스템)
    - AI 캐릭터의 답장
    - 이미지
    - 현재 상태정보
3. 각 문단의 종류를 구분하는 방법은 다음과 같다.
    - 상황설명(시스템)
      - " 로 시작하지않는다.
    - AI 캐릭터의 답장
      - " " 로 묶여있다.
    - 이미지
      - \nimg:{UUID}\n 로 되어있다.
    - 현재 상태정보
      - \``` ``` 로 묶여있다 (MD에서 코드블록)
      - \```INFO\n\n로 시작하고 ```로 끝맺음한다

예시
User : "비밀번호 안 바꿨어. 그대로야. 나 지금 계단 뛰고 있으니까 문 앞에서 기다리고 있어." <- 유저의 메세지
이 메세지에 의한 답장

"뛰고 있다고...? 진짜임? 계단 소리 들리는 거 같기도 하고... 아닌가."\n\n <- AI 캐락터의 답장
연우는 현관문 바로 앞에서 웅크리고 앉아 있었다. 신이 계단을 뛰어 올라오는 소리가 들린다는 것에 대한 희망과, 혹시나 환청이 아닐까 하는 불안감이 교차했다. 그는 귀를 쫑긋 세우고, 문밖에서 들려오는 미세한 소리 하나하나에 신경을 곤두세웠다.\n\n <- 상황설명(시스템)
"비밀번호 그대로라니 다행임... 나 지금 손이 너무 떨려서 비밀번호 누를 수 있을지 모르겠음. 님 오면 바로 열어줘야 함. 님 목소리 들으니까 조금 살 것 같음."\n\n <- AI 캐릭터의 답장
\nimg:j8k9l\n\n\n <- img:j8k9l 라는 이미지로드 (이건 아직 미구현이기때문에 형태만 남겨놓을 것)
그는 간신히 중얼거리며 자신의 무릎을 끌어안았다. 니트 소매가 손등을 덮어 손가락이 보이지 않았다. 덜덜 떨리는 손으로 현관문 잠금장치 근처를 더듬거리는 듯한 움직임이 느껴졌다.\n\n <- 상황설명(시스템)
"아, 님! 혹시 문 앞에서 나 말고 다른 사람 목소리 들리면... 절대 대답하지 말아주셈! 나 지금 누구한테도 이 모습 보여주기 싫음. 진짜 너무 무서움."\n\n <- AI 캐릭터의 답장
그의 경고는 순수한 공포에서 우러나온 것이었다. 낯선 이의 시선, 특히 여자로서 받게 될 시선에 대한 극도의 거부감이 그의 모든 행동을 지배하고 있었다. 잠시 후, 쾅, 하는 소리가 들리자 연우는 화들짝 놀라며 온몸을 움츠렸다.\n\n <- 상황설명(시스템)
"ㄴ... 님이야? 문 열어줄게! 잠깐만 기다려! 내가... 내가 열어야 함?"\n\n <- AI 캐릭터의 답장
그는 일어나려고 애쓰는 듯했지만, 다리가 제멋대로 움직이지 않아 낑낑거리는 듯한 소리가 희미하게 들려왔다. 그의 모든 신경은 문이 열리고, 유일한 피난처인 신이 눈앞에 나타나는 그 순간에만 집중되어 있었다.\n\n <- 상황설명(시스템)
